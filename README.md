# INTER + AÇÃO — Protótipo audiovisual

Experiência audiovisual curta para o formulário de Sociologia.

## Arquivos principais

- `index.html`
- `style.css`
- `script.js`

## Vídeos reais substituíveis

Para usar vídeos reais, coloque arquivos `.mp4` nesta pasta:

`assets/videos/`

com estes nomes:

- `ia.mp4`
- `redes-sociais.mp4`
- `enchente.mp4`

Se os vídeos não forem adicionados, a experiência usa cenas gráficas de fallback.

## Publicação

Suba todos os arquivos para o repositório GitHub Pages.

## Conceito

A aula é representada como a interseção entre dois compromissos:

- EU: preparação e apresentação de conteúdos, autores e conceitos.
- VOCÊ: preparação para participar das aulas.

Na interseção dos círculos surge a palavra `AULA`.
